# 🐥 Chick Escape

> 쿠키런 스타일의 2D 무한 러닝 액션 게임

---

## 🎮 게임 소개

플레이어가 **알 → 병아리 → 닭**으로 성장하며
점점 더 어려운 장애물을 극복하는 무한 러닝 게임.

---

## ✨ 주요 특징

- 🌾 모이를 먹으며 성장
- 🥚 알 → 🐤 병아리 → 🐔 닭 진화
- 🏃 슬라이딩 / 점프 / 2단 점프
- ❤️ 체력 시스템
- ⭐ 점수 시스템

---

## 🥚 성장 시스템

| 단계 | 사용 가능한 행동 | 새로운 장애물 | 진화 조건 |
|------|----------------|--------------|-----------|
| 🥚 알 | 1단 점프, 슬라이딩 | 🐍 뱀 | 모이 20개 + 점프 |
| 🐤 병아리 | 1단 점프, 슬라이딩 | 🪨 돌 | 모이 20개 + 점프 |
| 🐔 닭 | 2단 점프, 1단 점프, 슬라이딩 | 🌳 나무 | 최종 단계 |

---

## ❤️ 게임 오버

체력이 모두 소진되면 게임이 종료됩니다.

최종 화면에는

- ⏱ 플레이 시간
- ⭐ 점수
- 🌾 획득한 모이 개수

가 표시됩니다.
